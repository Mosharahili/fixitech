# fixitech

  A tech troubleshooting website with 50+ step-by-step guides for laptops, phones, Windows, Mac, Android, and iPhone issues.

  ## Stack
  - React 19 + Vite + Tailwind CSS v4
  - Wouter (routing), Framer Motion, shadcn/ui

  ## Getting Started
  ```bash
  npm install
  npm run dev
  ```

  ## Deploy to Vercel
  Import this repo on [vercel.com](https://vercel.com) — `vercel.json` handles SPA routing automatically.
